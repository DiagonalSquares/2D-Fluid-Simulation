name: feedback_commit_style
description: Commit messages should be concise and have no Co-Authored-By tag
type: feedback
---

- Keep commit messages short and to the point
- Do NOT include `Co-Authored-By: Claude` in commit messages

**Why:** User prefers brevity and doesn't want AI attribution visible in commit history.
**How to apply:** When committing, write a concise message with no co-author tag.
